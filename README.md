# Employee Management System

A professional desktop application for managing employee records with a modern Swing GUI, secure JDBC database connectivity, and comprehensive reporting features.

## 🚀 Features

### Core Features
- **Full CRUD Operations** - Create, Read, Update, Delete employee records
- **Secure Database Access** - JDBC with PreparedStatement to prevent SQL injection
- **Professional Swing GUI** - Material Design inspired interface with proper color scheme
- **Advanced Search** - Search employees by name, department, email, or designation
- **Charts Dashboard** - Visual analytics with department distribution and salary charts
- **CSV Export** - Export employee data with file chooser dialog
- **Transaction Management** - Commit/Rollback for data consistency
- **Input Validation** - Email format, phone number (10 digits), and salary validation

### User Interface
- Professional menu bar with File, Employee, Reports, and Help menus
- Toolbar with quick access buttons (Refresh, Search, Statistics, Export)
- Split pane layout with employee table and details form
- Status bar with real-time updates and system information
- Keyboard shortcuts for faster navigation

### Reports & Analytics
- **Statistics Dashboard** - View total employees, salary pool, average, min/max salary
- **Charts Dashboard** - Department distribution, average salary by department, employee count
- **Department Reports** - Detailed report with department-wise employee count and salary
